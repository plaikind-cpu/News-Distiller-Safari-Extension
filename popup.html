    📚 News-Distiller Help

    News-Distiller summarizes any news article and identifies its political lean using AI.

    How to use

    • Open any article in Safari, then tap the News-Distiller button.
    • Tap 🔍 Distill This Page — you'll get an Executive Summary and a Political Lean Assessment.
    • Or paste the article text into the box and tap 🔍 Distill This Text.

    Intended use

    Please use News-Distiller only with articles you're authorized to read — sites you subscribe to, gift links you've received, or freely-published content. Respect publisher paywalls.

    Distillations

    You initially get 5 distillations. After that, you'll need a friends-and-family access code to continue. Access codes are not available for purchase.

    Access codes

    If you have a News-Distiller iOS access code, enter it in the Access Code box and tap Save. iOS access codes start with NDi-. Codes for the desktop Chrome version don't work here, and vice versa.

    Full Report

    After distilling, tap 📄 Full Report to see a detailed view with Executive Summary, Key Findings, Implications, Context, and Political Lean Assessment.

    Saving as PDF

    Open the Full Report, then tap Safari's share icon → tap Options at the top → select PDF. From there you can Mail, Message, AirDrop, or save to Files.

    Privacy

    News-Distiller only sees the article text you ask it to analyze. Nothing else is read or stored from your browsing.
